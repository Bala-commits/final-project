# Retail Business Performance & Profitability Analysis

## Objective
The objective of this project is to analyze retail transactional data to identify profit-draining categories, optimize inventory turnover, and understand seasonal sales behavior.

## Tools Used
- SQL
- Python (Pandas, Seaborn)
- Tableau

## Project Workflow
1. Data cleaning and preprocessing using SQL
2. Profit margin analysis by category and sub-category
3. Inventory vs profitability correlation analysis using Python
4. Seasonal sales trend analysis
5. Dashboard creation using Tableau

## Deliverables
- SQL Queries (.sql file)
- Python Analysis Script (.py file)
- Tableau Dashboard
- PDF Report with insights

## Key Insights
- Some product categories generate low or negative profit margins
- Higher inventory holding days negatively impact profitability
- Sales follow seasonal patterns across months
